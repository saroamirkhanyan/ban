# Ban Programming Language
> Ban is programming language inspired by Armenian.

## Examples

```
գործառույթ կրկնիր կրկնվողգործառույթ և քանակ պարամետրերով
  (Անդրադարձման ավարտի պայման)
	եթե քանակ-ը և 0-ն հավասար
		պատասխանիր ոչինչ
	ավարտ
	կրկնվողգործառույթ քանակ
	(հաջորդ քայլ)
	սահմանիր հաջորդքանակ, որպես քանակ-ի և 1-ի տարբերություն-ը
	կրկնիր կրկնվողգործառույթ և հաջորդքանակ (Կանչ հաջորդ քայլը)
ավարտ

(
	f[0] = 1
	f[n] = n * f[n - 1]
)
գործառույթ ֆակտորիալ համար պարամետրերով
	եթե համար-ը և 0 հավասար
		պատասխանիր 1
	ավարտ
	սահմանիր նախորդհամար, որպես համար-ի և 1-ի տարբերություն-ը
	սահմանիր նախորդֆակտորիալ, որպես ֆակտորիալ նախորդհամար 
	պատասխանիր համար-ի և նախորդֆակտորիա-ի արտադրյալ-ը
ավարտ

գործառույթ տպիրֆակտորիալները կրկնություն պարամետրերով
	սահմանիր կրկնությանֆակտորիալ, որպես ֆակտորիալ կրկնություն
	տպիր կրկնությանֆակտորիալ
ավարտ

կրկնիր տպիրֆակտորիալները և 10
```

```
գործառույթ ապրիր ա և բ պարամետրերով
	եթե ա-ն և բ-ն հավասար
		տպիր 1
	ավարտ
	սահմանիր գ-ն, որպես ա-ի և բ-ի գումար-ը
	տպիր գ-ն
	սահմանիր դ-ն, որպես ա-ի և բ-ի տարբերություն-ը
	տպիր դ-ն 
	սահմանիր ե-ն, որպես ա-ի և բ-ի արտադրյալ-ը
	տպիր ե-ն
ավարտ

ապրիր 2 և 2
```

## Usage
```bash
git clone https://github.com/saroamirkhanyan/ban
cd ban
npm i
npm run build
node ./dist/ban.js [file.ban] 
```
